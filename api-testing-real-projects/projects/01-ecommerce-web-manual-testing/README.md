# Project 01 - Ecommerce Web Manual Testing

## Mục Đích Project
Xây dựng một dự án manual testing hoàn chỉnh cho ứng dụng web ecommerce để thể hiện năng lực QA cốt lõi.

## Ứng Dụng Được Kiểm Thử
Ứng dụng web (lĩnh vực ecommerce).

Ghi chú:
- Ứng dụng và module cụ thể sẽ được chốt khi bắt đầu triển khai chi tiết.

## Phạm Vi
- Requirement analysis
- Test plan
- Test scenarios
- Manual test cases
- Bug reports
- Test summary report

## Công Cụ
- Markdown
- Mẫu tài liệu kiểm thử
- Ảnh chụp bằng chứng (screenshots)

## Sẽ Bổ Sung Sau
- Phân tích yêu cầu theo module cụ thể
- Bộ test scenario/test case có ID truy vết
- Bug report thực tế kèm bằng chứng
- Test summary report cuối chu kỳ
