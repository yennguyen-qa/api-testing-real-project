# Project 04 - UI Automation with Playwright and TypeScript

## Mục Đích Project
Xây dựng nền tảng UI automation theo hướng ổn định, rõ ràng và có thể mở rộng.

## Ứng Dụng Được Kiểm Thử
Web UI mục tiêu sẽ được lựa chọn và mô tả trước khi bắt đầu viết test.

## Phạm Vi
- Thiết lập framework UI automation
- Viết test cho các user flow quan trọng
- Áp dụng Page Object Model
- Quản lý test data, report, evidence

## Công Cụ
- Playwright
- TypeScript
- Node.js
- Markdown

## Sẽ Bổ Sung Sau
- Bộ UI smoke test đầu tiên
- Cấu trúc page object tái sử dụng
- Chiến lược locator/assertion ổn định
- Tích hợp báo cáo test
