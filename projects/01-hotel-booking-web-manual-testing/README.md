# Project 01 - Hotel Booking Web Manual Testing

## Mục Đích Project
Xây dựng một dự án manual testing hoàn chỉnh cho ứng dụng web hotel booking để thể hiện năng lực QA cốt lõi.

## Ứng Dụng Được Kiểm Thử
Restful Booker Platform

- Web URL: https://automationintesting.online/
- Domain: Hotel / Room Booking

## Module Dự Kiến Kiểm Thử
- Home page
- Room listing
- Room booking
- Contact/message form
- Admin login và quản lý booking nếu nằm trong phạm vi học sau

## Phạm Vi
- Requirement analysis
- Test plan
- Test scenarios
- Manual test cases
- Bug reports
- Test summary report
- Evidence screenshots
- Jira traceability khi có bug

## Công Cụ
- Markdown
- Browser DevTools
- Jira
- Mẫu tài liệu kiểm thử
- Ảnh chụp bằng chứng (screenshots)

## Sẽ Bổ Sung Sau
- Phân tích yêu cầu theo module cụ thể
- Bộ test scenario/test case có ID truy vết
- Bug report thực tế kèm bằng chứng
- Jira key nếu phát hiện bug thật
- Test summary report cuối chu kỳ
