# Project 02 - API Manual Testing with Postman

## Mục Đích Project
Thể hiện năng lực kiểm thử API thủ công thông qua phân tích yêu cầu, thiết kế test, thực thi bằng Postman và quản lý evidence.

## API Được Kiểm Thử
Hệ thống REST API (sẽ xác định cụ thể ở bản cập nhật tiếp theo).

## Phạm Vi
- API requirement analysis
- API test plan
- API test scenarios
- API manual test cases
- Postman execution report
- Bug reports

## Công Cụ
- Postman
- Markdown
- JSON response evidence
- Screenshots

## Sẽ Bổ Sung Sau
- Ma trận endpoint và phạm vi kiểm thử
- Postman collection và environment thực tế
- Kết quả chạy test positive/negative
- Bug report API chi tiết
