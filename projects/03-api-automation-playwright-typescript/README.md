# Project 03 - API Automation with Playwright and TypeScript

## Mục Đích Project
Xây dựng nền tảng API automation có thể mở rộng và dễ bảo trì cho portfolio QA.

## API Được Kiểm Thử
API mục tiêu sẽ được chọn và mô tả rõ trước khi viết test chính thức.

## Phạm Vi
- Cấu hình Playwright cho API testing
- Cấu trúc API client tái sử dụng
- Quản lý test data, schema, utilities
- Mở rộng test smoke và regression theo từng giai đoạn
- Quản lý report và evidence

## Công Cụ
- Playwright
- TypeScript
- Node.js
- Markdown

## Sẽ Bổ Sung Sau
- Chiến lược environment config
- Bộ API smoke test đầu tiên
- Helper validate request/response
- Schema validation cơ bản
- Hướng dẫn chạy test và đọc report
